## üöÄ Getting started

Clone the project and access the folder.

```bash
$ git clone https://github.com/maarvins/calculator-js.git && cd calculator-js
```

Follow the steps below
```bash
# Install the dependencies
$ npm install


# Start the project
$ npm start
```
